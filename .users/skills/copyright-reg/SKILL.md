# Copyright Registration — 어문저작권 등록 서류 생성

## 개요

저작물 정보 → 업무상저작물 확인서 PDF + 저작권등록신청명세서 내용란 초안 생성

## 사전 조건

- `md-to-pdf` 글로벌 설치 (`npm i -g md-to-pdf`)
- 회사 정보: `docs-business/01. 회사 정보/base-info.md`, `team.md`
- 인감/서명: `docs-business/07.증명서/nextain-인감.png`, `양병석-서명.png`
- 로고: `about.nextain.io/public/assets/logos/nextain-light-logo.png`

## 워크플로우

### Step 1: 저작물 정보 파싱

사용자가 제공한 저작물 정보를 정리:

```
제호: (저작물 제목)
종류: 어문저작물 / 음악저작물 / 미술저작물 / 영상저작물 등
저작자: (주)넥스테인 (업무상저작물인 경우)
확인자: (실제 창작에 참여한 직원)
창작연월일: YYYY-MM-DD
```

회사 정보는 `docs-business/01. 회사 정보/base-info.md`, `team.md` 참조.

### Step 2: 업무상저작물 확인서 생성

템플릿: `docs-business/08.저작권/업무상저작물확인서_AI코딩가이드북.md`
CSS: `docs-business/08.저작권/확인서.css`

1. 템플릿을 복사하여 저작물 정보로 치환
2. md-to-pdf로 PDF 변환:

```bash
cd member-luke && npx md-to-pdf "docs-business/08.저작권/{출력파일}.md"
```

포함 요소:
- nextain 로고 (상단)
- 저작물 정보 테이블 (제호, 종류, 저작자)
- 확인 문구 + 발급일
- 법인인감 + "(주)넥스테인 대표이사 양병석"
- 확인자 정보 테이블 (성명, 직함, 서명, 전화번호, 주민등록번호, 주소)
- 안내문 (한국저작권위원회 공식 양식)
- 회사 정보 푸터

### Step 3: 저작권등록신청명세서 내용란 작성

GAI 활용 저작물인 경우 3구분 작성 (1000자 이내, 3000바이트):

```
① 신청물 전체 개요: 장르, 주제, 구성, 표현 요소 등
② 활용된 GAI 산출물: 사용 도구, 산출 과정, 역할 및 비중
③ 인간의 창작적 표현 부분: 구체적 창작 기여 항목 나열
```

참고: `docs-business/08.저작권/생성형 인공지능 활용 저작물의 저작권 등록 안내서.pdf`
- p.48-49 작성례 참조

### Step 4: 사용자 확인 (gate)

생성된 PDF + 내용란 초안을 사용자에게 보여주고 확인:
- "확인서 PDF와 내용란 초안을 확인해주세요."
- 수정 요청 시 반영 후 재생성

### Step 5: 출력 정리

출력 디렉토리: `docs-business/08.저작권/`

```
docs-business/08.저작권/
├── 확인서.css                          # 넥스테인 브랜딩 CSS (공용)
├── 업무상저작물확인서_{제호약칭}.md     # 확인서 소스
├── 업무상저작물확인서_{제호약칭}.pdf    # 확인서 PDF (제출용)
└── 생성형 인공지능 활용 저작물의 저작권 등록 안내서.pdf  # 참고
```

## 등록 절차 안내

1. **신청**: www.cros.or.kr (온라인) 또는 우편/방문
2. **제출물**: 저작권등록신청명세서(온라인 폼) + 복제물 PDF + 업무상저작물 확인서 PDF
3. **수수료**: 어문저작물 권리등록 약 24,000원
4. **시기**: 창작 후 1년 이내 등록 시 창작연월일 추정력 인정

## 주의사항

- GAI 산출물 자체는 등록 불가 — 인간의 창작적 기여가 있는 부분만 등록 가능
- 허위 등록 시 저작권법 제136조에 의해 처벌 가능 (3년 이하 징역 / 3천만원 이하 벌금)
- 업무상저작물 확인서의 확인자 = 실제 창작에 참여한 직원만 기재 (GAI 기재 불가)
- 주민등록번호 포함 문서이므로 git 커밋 시 주의 (PDF는 .gitignore 권장)
